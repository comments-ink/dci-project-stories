The stories project uses the backend and frontend of django-comments-ink. Comments can be nested down to two level of replies, and they display users' avatars. Comment reactions have been customized to resemble comment reactions in GitHub. In addition to comment reactions, there are object reactions: users can react to stories, in this project the only object reaction allowed is clapping. Comment voting is enabled too.

Threaded comments

Comments can be nested. The maximum thread level is customizable.

Notifications and mute-links

Receive follow-up notifications of your comments, and mute them later.

Comment confirmation

Comment confirmation via email when the user is not authenticated.

Comment reactions

The set of reactions is customizable, to allow more reactions other than like and dislike.

Object reactions

You may want to express that you liked a blog story by clicking on a clapping icon.

Comment votes

Vote comments up or down and list them by their score.
